# vue-eprase

> The Vue3 version of the Eprase client.

## About

The ePrescribing Risk and Safety Evaluation tool (ePRaSE) is designed to evaluate NHS Trust ePrescription services. The purpose of the tool is to encourage the correct use and optimisation of these systems to deliver improved patient outcomes. It is funded by NHS England and requires close working with the wider project team at Newcastle Hospitals NHS Foundation Trust and the wider team at NHS England.

The application has been in development since 2019 and has multiple repos, some of which are archived. The current application consists of a Vue3 SPA client and a Strapi back-end configured to work with Postgres.

Please visit the Vue-Eprase wiki page for in-depth documention.
